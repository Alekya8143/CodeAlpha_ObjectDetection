AI Object Detection & Tracking System

📌 Project Description

This project performs real-time object detection and tracking using YOLOv8 and OpenCV. It detects multiple objects from video streams and assigns tracking IDs while monitoring performance metrics.

🚀 Features

  • Real-time Object Detection
  • Object Tracking with Unique IDs
  • Object Counting
  • FPS Monitoring
  • YOLOv8 Deep Learning Model
