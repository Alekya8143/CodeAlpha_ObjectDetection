 AI Object Detection & Tracking System

 Overview
This project performs real-time object detection and tracking using YOLOv8 and OpenCV.

Features
- Real-time webcam detection
- Object tracking IDs
- Object counting
- FPS display

 Technologies Used
Python, OpenCV, YOLOv8

 Installation

Clone repository:
git clone https://github.com/Alekya8143/CodeAlpha_ObjectDetection.git

Install dependencies:
pip install -r requirements.txt

Run project:
python app.py

 Controls
Press Q → Exit
Press S → Save Screenshot
